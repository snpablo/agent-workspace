# Exec Plans

This folder holds structured execution plans for work in this repository.

## Structure

- `active/`: plans currently in motion or under active consideration
- `completed/`: plans kept for historical reference
- `templates/`: reusable plan formats

## Naming Convention

Use numeric prefixes so plans read in sequence:

- `01-model-objects.md`
- `02-output-architecture.md`
- `03-workspace-architecture.md`

This is preferred over day-of-week naming because the order remains useful even when work shifts or some items are skipped.
